Olá, seja bem vindo!

Este projeto foi criado para reunir miniprogramas escritos na linguagem de programação C. Os exemplos aqui mostrados foram retirados de pesquisas em sites de universidades do Brasil inteiro e também das listas de exercícios disponibilizadas pelo professor da disciplina de Programação e Estruturas de Dados do Instituto de Educação Ciência e Tecnologia da Paraíba - IFPB no periodo 2011.1 para os alunos do Curso Sistemas para Internet.

Espero que você perceba alguns detalhes da linguagem, aprenda e comente (construtivamente) os códigos postados aqui.

Obrigado!
